Reflexión

¿Cuál es la diferencia entre padding y margin?

El padding es el espacio interno entre el contenido y el borde del elemento; forma parte del elemento y tiene su mismo color de fondo. El margin es el espacio externo que separa al elemento de otros elementos; es transparente y no pertenece al elemento en sí.

¿Por qué se recomienda usar box-sizing: border-box?

Con border-box, el padding yel border quedan incluidos dentro del ancho total del elemento, por lo que el tamaño que declaramos es exactamente el tamaño final. Eso facilita el diseño y evita desbordamientos inesperados.

¿Qué significa diseñar con enfoque mobile-first?

Significa escribir primero los estilos base pensados para pantallas pequeñas (móviles) y despues usas @media queries con min-width para ajustar estilos conforme la pantalla crece (tablet, desktop). Este enfoque prioriza la experiencia del usuario mediante una escabilidad de dispositivos.

¿En qué situaciones preferirías rem sobre px?

Se usa mas rem para textos y espaciados porque se adapta en el tamaño de fuente raíz del documento (html). En cambio, el uso de px para valores que no deben escalar, como bordes de 1px o sombras.
